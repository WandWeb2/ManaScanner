## ğŸ“– Quick Start Guide

### First Time Setup

1. **Launch ManaScanner** - Open the application using your preferred method
2. **Create Your Collection** - Start by setting up your first collection
3. **Scan Your Cards** - Use the camera feature to scan Magic: The Gathering cards
4. **Organize** - Create decks and organize your collection
5. **Track Values** - Monitor the market value of your cards

### Basic Usage

#### Scanning Cards

1. Click on the **Scan** button
2. Position your card in front of the camera
3. Wait for automatic recognition
4. Confirm or edit card details
5. Add to your collection

#### Managing Collections

1. Navigate to **Collections**
2. Create a new collection or select an existing one
3. Add cards manually or via scanning
4. View collection statistics and total value

#### Building Decks

1. Go to **Deck Builder**
2. Create a new deck
3. Search for cards or add from your collection
4. Save and manage your deck lists

#### MTG Arena Integration

For MTG Arena players, we provide an automatic deck monitoring daemon:

1. Navigate to the **mtg-arena-daemon** directory
2. Follow the instructions in the [MTG Arena Daemon README](mtg-arena-daemon/README.md)
3. The daemon will automatically export your Arena decks for use with ManaScanner

## ğŸ› Troubleshooting

### Common Issues

**Issue: Application won't start**
- Ensure all prerequisites are installed
- Check that you're using the correct version for your platform
- Try running from the command line to see error messages

**Issue: Card scanning not working**
- Grant camera permissions to the application
- Ensure adequate lighting for card recognition
- Make sure the card is in focus and clearly visible

**Issue: Can't find specific cards**
- Check your internet connection (needed for card database)
- Verify the card set and edition
- Try scanning the card again with better lighting

### Getting Help
